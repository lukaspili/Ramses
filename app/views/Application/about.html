L'application Eunomie a été réalisée par le laboratoire Java de SUPINFO Paris afin de répondre aux besoins du campus.
Cette application permet de gérer les STA, les contrats cadres, les SOE et les bons de commande.

Les technologies utilisées sont bien evidemment Java, avec Play!Framework, JQuery et PostgreSQL pour la persistence des données. L'hébergement est assuré par la plateforme cloud Heroku et le stockage des contrats et des bons de commande par Amazon S3.
